# Flight Simulator Project

This flight simulator is a simple project I made in my free time. It does not depict real-life scenarios and is currently unfinished. The project aims to provide a basic simulation experience for educational purposes.

## Features

- Basic flight controls
- Google earth photorealistic data
- A simple main menu where you can change graphics settings.
